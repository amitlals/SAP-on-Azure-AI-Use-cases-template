# Use Cases

This directory contains detailed documentation and implementation guides for each SAP on Azure AI use case.

## üìÅ Available Use Cases

### 1. [Screenshot to Fiori App](01-screenshot-to-fiori/)
Transform UI screenshots into functional Fiori applications in under 5 minutes using AI assistance.

**Key Technologies**: SAP Fiori, GitHub Copilot, UI5
**Difficulty**: Intermediate
**Time to Complete**: ~5-10 minutes

### 2. [Dual Copilot Integration](02-dual-copilot/)
Leverage the power of both SAP Joule for Developers (J4D) and GitHub Copilot for enhanced development productivity.

**Key Technologies**: J4D, GitHub Copilot, SAP BAS
**Difficulty**: Beginner to Intermediate
**Time to Complete**: ~15-20 minutes

### 3. [Fiori App Development with Joule](03-fiori-with-joule/)
Build Fiori applications using SAP Joule's AI-powered assistance for code generation and best practices.

**Key Technologies**: SAP Fiori, Joule, SAPUI5
**Difficulty**: Intermediate
**Time to Complete**: ~20-30 minutes

### 4. [SAP Build Low-Code Solutions](04-sap-build-lowcode/)
Create enterprise applications using SAP Build's low-code/no-code platform with Joule Copilot assistance.

**Key Technologies**: SAP Build, Joule Copilot, Low-Code
**Difficulty**: Beginner
**Time to Complete**: ~15-25 minutes

### 5. [Azure SRE Agent Integration](05-azure-sre-agent/)
Implement agentic DevOps with Azure SRE Agent for SAP workloads running on Azure.

**Key Technologies**: Azure SRE Agent, SAP on Azure, DevOps
**Difficulty**: Advanced
**Time to Complete**: ~30-45 minutes

### 6. [SAP Joule AMA (Ask Me Anything)](06-joule-ama/)
Explore SAP Joule's conversational AI capabilities for querying SAP systems and data.

**Key Technologies**: SAP Joule, Natural Language Processing
**Difficulty**: Beginner
**Time to Complete**: ~10-15 minutes

### 7. [SAP-RPT-1-OSS Tabular LLM](07-sap-rpt-oss/)
Work with SAP's open-source tabular LLM for structured data processing and analysis.

**Key Technologies**: SAP RPT-1-OSS, Python, ML
**Difficulty**: Advanced
**Time to Complete**: ~45-60 minutes

### 8. [Foundry Local - On-Device AI](08-foundry-local/)
Run AI models locally on your device with complete data privacy, no Azure subscription required.

**Key Technologies**: Foundry Local, On-Device AI, Privacy-First
**Difficulty**: Intermediate
**Time to Complete**: ~20-30 minutes
